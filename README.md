# HTML-and-CSS-Projects

This repository contains two major web development projects demonstrating proficiency in modern front-end technologies.

## 🚀 Projects Included

### 1. Cinema Booking Website (Bootstrap)
A responsive movie theater website built using **Bootstrap 5**. 
- **Features:**
  - Navigation with HTML bookmarks for smooth scrolling.
  - Interactive "Now Showing" section using Bootstrap Card Decks.
  - Use of **Info Pill Badges** for displaying movie showtimes.
  - Fully responsive design for mobile and desktop viewing.

### 2. Advanced CSS Portfolio
A unique, colorful personal portfolio that utilizes core HTML5 and CSS3 features.
- **Key Technical Implementations:**
  - **Fixed Navigation:** A sticky menu with hover effects.
  - **Video Integration:** A background video header that autoplays on mute and loops.
  - **CSS Manipulation:** Centered images with rounded edges and `hue-rotate` filters.
  - **Overlays:** Image fade-in effects and text-over-video layouts.
  - **Data Presentation:** A styled table with border-collapse and specific row coloring.
  - **Layouts:** A custom two-column layout with fixed dimensions.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Custom Styles & Animations)
- Bootstrap 5
- GitHub for Version Control
